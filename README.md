# Logistics Service API

REST API untuk Logistics Management Service menggunakan Go.

## Fitur
- Create Shipment
- Update Shipment Status
- Track Shipment
- Get Shipments

## Instalasi
1. Clone repository:
```bash
git clone https://github.com/elsanoviani/logistics-service.git
cd logistics-service
```
2. Clone repository:
```bash
go mod tidy
```

3. Jalankan server:
```bash
go run main.go
```
